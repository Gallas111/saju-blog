## 본인 정관 편관이 본인 본인의 호흡에 보내는 신호

본인 십신(十神)은 본인 본인의 사주에서 본인 본인의 일간과 본인 본인의 다른 7글자(천간 3 + 지지 4)의 본인 본인의 관계를 본인 본인의 10가지 본인 본인의 자가 호흡 자원으로 본인 본인의 호흡에 본인 본인의 자가 단서로 본인 들어오는 본인 본인의 자가 호흡 자원이에요. 본인 본인의 10가지 본인 본인의 자가 호흡 자원은 본인 본인의 비견·겁재(나)·식신·상관(자녀)·편재·정재(재물)·편관·정관(권위)·편인·정인(부모) 본인 10개 본인 본인의 자가 호흡 자원이에요.

본인 정관 편관 본인 관성은 본인 본인의 일간을 본인 본인의 자가 호흡 자원으로 본인 본인의 극(克)하는 본인 본인의 5행 본인 본인의 자가 단서로 본인 본인의 호흡에 본인 본인의 자가 단서로 본인 들어와요. 본인 본인의 일간 갑·을(목)이면 본인 본인의 관성은 본인 경·신(금) 본인 본인의 자가 호흡 자원이고, 본인 본인의 일간 병·정(화)이면 본인 본인의 관성은 본인 임·계(수) 본인 본인의 자가 호흡 자원이고, 본인 본인의 일간 무·기(토)이면 본인 본인의 관성은 본인 갑·을(목) 본인 본인의 자가 호흡 자원이에요.

본인 정관(正官)은 본인 본인의 일간과 본인 본인의 음양이 본인 본인의 다른 본인 본인의 자가 관성 본인 본인의 자가 호흡 자원으로 본인 본인의 정상적 권위·본인 본인의 안정 본인 본인의 직장·본인 본인의 책임이 본인 본인의 호흡에 본인 본인의 자가 단서로 본인 들어와요. 본인 편관(偏官)은 본인 본인의 일간과 본인 본인의 음양이 본인 본인의 같은 본인 본인의 자가 관성 본인 본인의 자가 호흡 자원으로 본인 본인의 강한 권위·본인 본인의 도전·본인 본인의 군경·본인 본인의 사업이 본인 본인의 호흡에 본인 본인의 자가 단서로 본인 들어와요.

본인 본인의 사주 십신 본인 본인의 자가 호흡 자원 전체 본인 본인의 자가 풀이는 [십성(십신) 완벽 정리 본인 사주 읽는 법](/blog/sipsung-ten-gods-guide)에서 본인 손으로 함께 보세요. 본인 정관 편관 본인 본인의 자가 차이 본인 풀이는 [사주 정관 편관 차이 십신 관성 7가지 작용과 직업운 풀이](/blog/jeonggwan-pyeongwan-difference-saju-officer-stars)에서 본인 손으로 함께 보세요.

## 본인 정관 편관 7유형 — 본인 본인의 자가 단서

본인 정관 편관 7유형은 본인 본인의 관성 본인 본인의 자가 호흡 자원이 본인 본인의 일간 + 본인 본인의 위치(연·월·일·시주) + 본인 본인의 강약(많음·적음·없음) 본인 본인의 자가 단서로 본인 본인의 호흡에 본인 본인의 자가 호흡 자원으로 본인 들어와요.

**1유형 — 본인 정관 본인 본인의 자가 단서(정상적 권위)**
본인 본인의 사주에 본인 정관이 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 1~2개 본인 본인의 호흡에 본인 본인의 자가 단서로 들어오면 본인 본인의 자가 정상적 권위·본인 본인의 안정 본인 본인의 직장(공무원·대기업·교사·은행)·본인 본인의 책임 본인 본인의 자가 호흡 자원이 본인 본인의 호흡에 본인 본인의 자가 단서로 본인 들어와요. 본인 본인의 직장 상사와의 본인 본인의 자가 관계는 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 협력 호흡이 본인 본인의 자가 호흡 자원이에요.

**2유형 — 본인 편관 본인 본인의 자가 단서(강한 권위·도전)**
본인 본인의 사주에 본인 편관이 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 1~2개 본인 본인의 호흡에 본인 본인의 자가 단서로 들어오면 본인 본인의 자가 강한 권위·본인 본인의 도전·본인 본인의 군경·본인 본인의 사업·본인 본인의 자영업·본인 본인의 운동 본인 본인의 자가 호흡 자원이 본인 본인의 호흡에 본인 본인의 자가 단서로 본인 들어와요. 본인 본인의 직장 상사와의 본인 본인의 자가 관계는 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 충돌 호흡 또는 본인 본인의 자가 추월 호흡이 본인 본인의 자가 호흡 자원이에요.

**3유형 — 본인 정관 편관 본인 본인의 자가 혼잡(관살혼잡 官殺混雜)**
본인 본인의 사주에 본인 정관 + 편관이 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 함께 본인 본인의 호흡에 본인 본인의 자가 단서로 들어오면 본인 본인의 자가 권위 혼잡 호흡 본인 본인의 자가 호흡 자원이에요. 본인 본인의 직장 상사가 본인 본인의 자가 호흡 자원으로 본인 본인의 다중 본인 본인의 자가 흐름(2명·3명)으로 본인 본인의 호흡에 본인 본인의 자가 단서로 본인 들어오고 본인 본인의 자가 호흡 자원이 본인 본인의 자가 분산 호흡이라서 본인 본인의 의식적 본인 본인의 자가 호흡 자원으로 본인 본인의 우선순위 호흡이 본인 본인의 자가 호흡 자원이에요.

**4유형 — 본인 정관 편관 본인 본인의 자가 무관성(無官星)**
본인 본인의 사주에 본인 정관 편관이 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 없으면 본인 본인의 자가 무관성 호흡 본인 본인의 자가 호흡 자원이에요. 본인 본인의 직장 상사 본인 본인의 자가 호흡 자원이 본인 본인의 호흡에 본인 본인의 자가 단서로 본인 약하게 본인 들어와서 본인 본인의 자가 호흡 자원이 본인 본인의 자유 호흡(프리랜서·자영업·1인 사업·예술)이 본인 본인의 자가 호흡 자원이에요.

**5유형 — 본인 정관 본인 본인의 자가 강세(많음 3개+)**
본인 본인의 사주에 본인 정관이 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 3개 이상 본인 본인의 호흡에 본인 본인의 자가 단서로 들어오면 본인 본인의 자가 권위 과다 호흡 본인 본인의 자가 호흡 자원이에요. 본인 본인의 책임이 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 과중 호흡으로 본인 본인의 호흡에 본인 본인의 자가 단서로 본인 들어와요. 본인 본인의 의식적 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 위임 호흡 + 본인 본인의 자가 휴식 호흡이 본인 본인의 자가 호흡 자원이에요.

**6유형 — 본인 편관 본인 본인의 자가 강세(많음 3개+)**
본인 본인의 사주에 본인 편관이 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 3개 이상 본인 본인의 호흡에 본인 본인의 자가 단서로 들어오면 본인 본인의 자가 도전 과다 호흡 본인 본인의 자가 호흡 자원이에요. 본인 본인의 자가 호흡 자원이 본인 본인의 자가 충돌 호흡 + 본인 본인의 자가 위험 호흡(사고·소송·갈등)으로 본인 본인의 호흡에 본인 본인의 자가 단서로 본인 들어와요. 본인 본인의 의식적 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 신중 호흡 + 본인 본인의 자가 식상(자녀·창의·발산) 본인 본인의 자가 보강 호흡이 본인 본인의 자가 호흡 자원이에요.

**7유형 — 본인 정관 편관 본인 본인의 자가 위치(연·월·일·시주)별 흐름**
본인 본인의 정관 편관이 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 어느 위치(연주·월주·일주·시주)에 본인 본인의 호흡에 본인 본인의 자가 단서로 들어오는지 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 흐름 본인 본인의 자가 호흡 자원이에요. 본인 본인의 연주 본인 본인의 자가 호흡 자원이면 본인 본인의 초년·부모 본인 본인의 자가 호흡 자원이고, 본인 본인의 월주 본인 본인의 자가 호흡 자원이면 본인 본인의 청년·직장 본인 본인의 자가 호흡 자원이고, 본인 본인의 일주 본인 본인의 자가 호흡 자원이면 본인 본인의 본인 본인의 자가 호흡 자원이고, 본인 본인의 시주 본인 본인의 자가 호흡 자원이면 본인 본인의 자녀·말년 본인 본인의 자가 호흡 자원이에요.

## 본인 정관 편관 자가 점검 7단계 — 본인 손으로 풀기

본인 정관 편관 7유형을 본인 손으로 점검하는 본인의 자가 7단계 순서를 본인이 본인 본인의 호흡으로 따라 해보세요.

**1단계 — 본인 본인의 사주명식 본인 손으로 본인 본인의 자가 호흡 자원으로 본인 뽑기.** 본인의 양력 생년월일·시·성별 4가지 정보로 본인 만세력 앱에서 본인 본인의 사주 8글자(천간 4 + 지지 4)를 본인 손으로 확인하세요.

**2단계 — 본인 본인의 일간 본인 본인의 자가 호흡 자원으로 본인 손으로 본인 본인의 자가 단서로 본인 확인.** 본인 본인의 일주 천간 1글자(갑·을·병·정·무·기·경·신·임·계 중 1개)가 본인 본인의 일간이에요. 본인 본인의 일간을 본인 손으로 본인 본인의 자가 호흡 자원으로 본인 확인하세요.

**3단계 — 본인 본인의 관성 본인 본인의 자가 호흡 자원으로 본인 손으로 본인 본인의 자가 단서로 본인 식별.** 본인 본인의 일간을 본인 본인의 자가 호흡 자원으로 본인 본인의 극(克)하는 본인 본인의 5행 본인 본인의 자가 단서를 본인 손으로 본인 본인의 자가 호흡 자원으로 본인 본인의 자가 식별하세요. 본인 본인의 일간 갑·을(목)이면 본인 본인의 관성은 본인 경·신(금) 본인 본인의 자가 호흡 자원이에요.
